# Skill Registry: govcheck

## Project Standards

### Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (framer-motion)
- **Auth**: Action-based (mock)

### Conventions
- **Components**: Functional components with TypeScript interfaces.
- **Server Actions**: Used for data mutations and auth.
- **Hooks**: Custom hooks for logic separation (e.g., `useVehicleSearch`).
- **UI**: Glassmorphism, Dark Mode, premium aesthetic.

## Compact Rules

### React/Next.js
- Use `use client` only when necessary.
- Prefer Server Components for data fetching.
- Use `lucide-react` for icons.

### Tailwind 4
- Use modern Tailwind 4 syntax.
- Follow the established glassmorphism patterns.
